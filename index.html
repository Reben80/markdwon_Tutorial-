What is Markdown?

  • Lightweight markup language
  • Created by John Gruber in 2004
  • Easy-to-read, easy-to-write plain text format
  • Converts to structurally valid HTML

Why Markdown?

  • Markdown's versatility makes it a valuable tool in educational settings, both for students and faculty.
  • Whether you're creating course materials, documenting a project, or organizing study notes, building a website, Markdown provides a simple, efficient way to format and share content.

print(greet("World"))

Blockquotes

Simple Blockquote

Darkness cannot drive out darkness, only light can do that. Hate cannot drive out hate, only love can do that.

— Martin Luther King Jr.

Markdown Syntax:

> Darkness cannot drive out darkness, only light can do that. Hate cannot drive out hate, only love can do that.
>
> — Martin Luther King Jr.

Nested Blockquotes

The best way to predict the future is to create it.

It's hard to make predictions, especially about the future.

— Yogi Berra
— Peter Drucker

Markdown Syntax:

> The best way to predict the future is to create it.
>
>> It's hard to make predictions, especially about the future.
>>
>> — Yogi Berra
>
> — Peter Drucker

Blockquotes with Other Elements

The quarterly results look great!

  • Revenue was off the chart.
  • Profits were higher than ever.

Everything is going according to plan.


Markdown Syntax:

> #### The quarterly results look great!
>
> - Revenue was off the chart.
> - Profits were higher than ever.
>
> *Everything* is going according to **plan**.

Popular Markdown Editors

Visual Studio Code

  • Free, open-source, highly customizable
  • Great for developers who also work with code
  • Supports Markdown extensions for enhanced functionality

Obsidian

  • Free for personal use, focuses on linking between notes
  • Great for creating knowledge bases or zettelkasten systems
  • Supports plugins for extended functionality

Typora

  • Clean, distraction-free interface with live preview
  • Seamless transition between writing and previewing
  • Supports custom themes and CSS

MarkText

  • Open-source, cross-platform with a simple interface
  • Supports CommonMark Spec and GitHub Flavored Markdown
  • Offers both source code mode and preview mode

Joplin

  • Free, open-source, focuses on note-taking and to-do lists
  • Supports end-to-end encryption for secure note-taking
  • Synchronization across multiple devices

Limitations of Markdown

No Native Support for Text Colors

Standard Markdown text

This text is red using HTML

Note: HTML color may not work in all Markdown processors

Standard Markdown text

This text is red using HTML

*Note: HTML color may not work 
in all Markdown processors*

Limited Layout Control

  • No built-in way to create multi-column layouts
  • Limited control over image sizing and positioning
  • No native support for page breaks or complex formatting

Lack of Standardization

  • Different Markdown flavors may have slightly different syntax
  • Some advanced features may not be universally supported
  • Rendering can vary between different Markdown processors

Limited Support for Complex Tables

  • No native support for merged cells or complex table layouts
  • Limited options for table styling
  • Large tables can become unwieldy in raw Markdown
